Why understanding HTML CSS and JavaScript is important

If you are interested in becomming a Web Developer it is crucial to have a thorough understanding of HTML, CSS and JavaScript.

HTML Basics

HTML is an acronym for Hyper Text Markup Language, and is the standard markup language used for creating web pages.

HTML allows for the creation and structure of sections, links, and paragraphs using elements, and is also most commenly known as the building blocks of a web page.

  • HTML consists of a series of both elements and attributes.
  • HTML elements are represented by < > tags

CSS Basics

CSS is an acronym for Cascading Style Sheets, and is used for styling elements written in a markup language such as HTML.

CSS is most commenly used with HTML, but is also used with other markup languages such as SVG or XML.

  • CSS describes how HTML elements should be dislayed on paper, screen, or in other media.
  • In CSS a class selector is formatted as a period(.) and an id selector is formatted by a hash(#).
  • CSS is used for styling the way a web page looks and feels. With CSS you are able to control the style of fonts, the spacing between paragraphs, what background images or colors are used as well as a variety of other effects.

JavaScript Basics

JacaScript is a programming language that works alongside HTML and CSS, and adds interactivity to a website.

JavaScript is uded to add interactive elements to a web page that engage users.

  • JavaScript is a scripting language that is used to create dynamic Web Page content.
  • JavaScript is used to create elements for improving site visitors interaction with web pages, such as animated graphics, dropdown menus, and dynamic background colors.
  • JavaScript selectors are used to find (select) HTML elements based on their tag name, classes, id, types, attributes and many more.
  • examples of the most common JavaScript selectors are the following, getElementById, querySelector, and querySelectorAll

How HTML CSS and JavaScript work together

HTML, CSS, and JavaScript build on one another from the most simple website structure to the most advanced interactive features.

HTML is the structure of a websites content, while CSS adds style and formatting to these structures. JavaScript turns all of those stylized components into a final product that a user can interact with.

HTML CSS and JavaScript are all very important building blocks when it comes to creating a website. Each technology has its own specific purpose but when used together they can create fully functional web page.
